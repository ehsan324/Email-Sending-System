What is the maximum number of emails sent per call?

The maximum number of simultaneous sending and receiving statuses per call is 200.

Do we have to pay a fee to send from the web service?

Yes, for this purpose, go to the line management section and set the desired line. In this section, you can specify the address for transferring received SMS messages as well as the GET / POST call method.

Is it possible to transfer received SMS messages to a URL?

Yes, as soon as we receive this status from the telecom, the record status will be updated and the fee will be refunded to your account.

Some SMS messages return a status of "Not delivered to phone" or "Error". Do these SMS messages have a way to reach their destination?

Yes, one of the features of IRAN ESS web service is to create such triggers. For this purpose, please go to advanced settings and enable resending of SMS messages. Note that this feature is enabled for the advanced web service version..

Can purchased lines be transferred to other systems?

Yes, this is possible with a written letter.

How is the profit from charging customers calculated?

For more information, visit Customer Management > Tariff Management.
